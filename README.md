# Facebook Sentiment Analysis Flask App

A Flask-based web application that analyzes Facebook comments and detects whether the comment contains **hate speech or non-hate speech** using a trained Machine Learning model.

---

## 🚀 Features

- User Registration and Login System
- User Dashboard
- Facebook Comment Sentiment Analysis
- Hate Speech Detection using Machine Learning
- Comment History Tracking
- Secure Environment Variables (.env)
- MySQL Database Integration

---

## 🧠 Machine Learning Model

The project uses Natural Language Processing (NLP) techniques to detect hate speech.

Model files used:

- hate_speech_text_model.pkl
- hate_speech_tfidf.pkl

These models are loaded using **joblib** for real-time prediction.
