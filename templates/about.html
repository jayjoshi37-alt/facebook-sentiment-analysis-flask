About SentimentAI

SentimentAI is an advanced AI-powered sentiment and hate speech detection platform designed to analyze Facebook comments in real-time using Machine Learning and NLP.

Our Mission

Our mission is to empower brands and businesses to understand their audience emotions using AI-driven insights. We help detect:

  • ✔ Hate Speech Detection
  • ✔ Emoji Sentiment Analysis
  • ✔ Real-time Comment Processing
  • ✔ Advanced NLP-based Classification

Technology Stack

  • • Python (Flask Backend)
  • • Scikit-Learn ML Model
  • • TF-IDF Vectorization
  • • MySQL Database
  • • TailwindCSS Frontend
